# Todo Application

A feature-rich todo application with local storage persistence.

## Features
- Add, edit, and delete tasks
- Mark tasks as complete/incomplete
- Filter tasks (All, Pending, Completed)
- Local storage persistence
- Responsive design
- Task statistics

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API
- Font Awesome Icons

## Setup
1. Clone the repository
2. Open `index.html` in your browser
3. No build process required

## Usage
- Type a task and click "Add Task" or press Enter
- Click checkbox to mark complete/incomplete
- Use filter buttons to view different task categories
- Edit tasks using the Edit button
- Clear completed tasks or all tasks using action buttons
